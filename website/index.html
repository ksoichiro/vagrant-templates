Custom Vagrant boxes

Usage

Download or copy the following links and execute the commands below.

  $ vagrant box add TITLE URL
or if you've already downloaded the box file,
  $ vagrant box add TITLE /path/to/boxname.box
then
  $ vagrant init TITLE
  $ vagrant up
