vagrant-templates
=================

Vagrantfile and provisioning scripts templates.

Some of the boxes are provided by my [website](http://files.sika524.com/box/).

## Requirements

Scripts in this project are tested on the following environment.

* VirtualBox(4.3.10)
* Vagrant (1.5.1)
